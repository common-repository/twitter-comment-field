This plugin adds a twitter username field to the comments form. Follow buttons are then displayed for the comments.

== Description ==

Twitter Comment Field (TCF) adds a twitter username field to the comments form of your theme. If a username is entered, a custom twitter follow button is displayed for that commentator. If left blank, no code will be added to that comment.

== Installation ==

1. Upload `twitter-comment-field` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
